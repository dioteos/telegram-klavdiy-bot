Перевір наявність оновлень для Telegram плагіна Claude Code (telegram@claude-plugins-official).

Кроки:
1. Дізнайся встановлену версію: `ls ~/.claude/plugins/cache/claude-plugins-official/telegram/`
2. Дізнайся останню версію: `npm view claude-channel-telegram version`
3. Порівняй встановлену з останньою

Якщо є нова версія — відправ повідомлення з номерами версій і командою для оновлення.

Якщо версії збігаються — НЕ надсилай нічого, просто заверши мовчки.
